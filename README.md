# QuickServeIT - Premium Web Solutions

A modern, fully static frontend application for QuickServeIT's digital agency services. Built with React, Vite, and Tailwind CSS.

## Features

- 🎨 **Premium UI/UX Design** - Luxury-focused responsive design
- ⚡ **Ultra-fast Performance** - Static site deployed on Vercel's CDN
- 🔒 **100% Client-Side** - No backend dependencies or server needed
- 📱 **Mobile Optimized** - Fully responsive across all devices
- 🎬 **Smooth Animations** - Framer Motion animations and particle effects
- 📊 **SEO Optimized** - Meta tags and semantic HTML

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + CSS Variables
- **Animations**: Framer Motion
- **UI Components**: Radix UI
- **Form Handling**: React Hook Form + Zod
- **Icons**: Lucide React
- **Hosting**: Vercel

## Services

1. **Institute Services** - Educational documentation and scholarship processing
2. **Creative Studio** - Video production, editing, and motion graphics
3. **Portfolio** - Professional work samples and case studies

## Project Structure

```
├── client/                 # Main React frontend
│   ├── src/
│   │   ├── pages/         # All website pages
│   │   ├── components/    # Reusable UI components
│   │   ├── hooks/         # Custom React hooks
│   │   ├── lib/           # Utilities and helpers
│   │   └── App.tsx        # Main router configuration
│   └── public/            # Static assets
├── backend-archive/       # Preserved backend code (not deployed)
├── shared/                # Type definitions
└── config files           # Vite, Tailwind, TypeScript configs
```

## Getting Started

### Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run start
```

## Deployment

Deployed on Vercel - fully automated from GitHub commits.

- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Framework**: Vite (auto-detected)
